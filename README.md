# AWT Cargo & Couriers Website

A modern, responsive website for AWT Cargo & Couriers Solution Pvt. Ltd. built with Python Flask.

## Features

- ✅ Online parcel booking with tracking number generation
- ✅ Real-time parcel tracking with status updates
- ✅ Rate calculator with real-time pricing
- ✅ Partner registration system
- ✅ Customer dashboard
- ✅ Responsive design with Bootstrap 5
- ✅ Multilingual support ready (English + Marathi)
- ✅ Contact forms with email integration
- ✅ Service information pages
- ✅ SQLite database for data storage

## Technology Stack

- **Backend**: Python with Flask
- **Frontend**: HTML5, CSS3, JavaScript, Bootstrap 5
- **Database**: SQLite (development) / PostgreSQL (production)
- **Deployment**: Render.com ready
